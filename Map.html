Instructions/ user guide

Keyboard keys:

  • left - move left
  • right - move right
  • up - move up
  • down - move down
  • t - move towards you (Zoom in)
  • shift - move away from you (Zoom out)
  • x - rotate x axis
  • y - rotate y axis
  • z - rotate z axis
  • a - rotate x axis (but other direction)
  • b - rotate y axis (but other direction)
  • c - rotate z axis (but other direction)

Approach

features

I have attempted to implement the following features:

  • Basic 3D shapes
  • Displaying simple shapes using either a wireframe/flat surface representation
  • A means of projection (a perspective view)
  • Hidden Surface removal - Vector approach (Ok - it works, although i haven't implemented
    removal of hidden surface ambiguities)
  • Shading: used filled polygons to represent the surface
  • Implemented Lighting - Lambert Shading
  • Viewpoint Transformations (Zooming in & out)
  • Multi-threaded Double Buffering
  • User control of transformations using keyboard keys

I therefore believe that the assignment fall in the 80% - 100% category.
